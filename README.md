## Deskripsi Singkat
SENTRI adalah aplikasi AI berbasis chatbot yang dirancang untuk membantu masyarakat di wilayah Indonesia Timur memahami dan mengelola keuangan digital, menghindari penipuan, dan mendapatkan edukasi keuangan secara mudah dan interaktif.

## Fitur Utama
- Asisten Chatbot AI edukatif (bahasa Indonesia dan lokal)
- Deteksi potensi penipuan (fraud detection)
- Panduan kerja, tabung, dan investasi
- Edukasi keuangan via tanya jawab

## Teknologi yang Digunakan
- ChatGPT API / OpenAI
- Python (Flask / Streamlit) atau HTML-JS (jika web)
- Dataset publik untuk pelatihan fraud detection
- Hosting di Replit / Netlify / GitHub Pages (sementara)
